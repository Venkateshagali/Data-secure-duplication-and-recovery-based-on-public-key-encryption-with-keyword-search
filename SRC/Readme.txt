Data secure de-duplication and recovery based on public key encryption with keyword search

In this project have 4 modules

1.Data Owner
2.Data User
3.Test Authority
4.Cloud Server Provider


1.Data Owner

* Register 
* After authorize by cloud owner login the account 
* Upload the file with the encryption format.
* View user request and Make a request for re-encryption
* View the status and Re-Encrypted the key
* Logout

2.Data User

* Register 
* After authorize by cloud user login the account 
* Profile
* Search file make a Request for downloading the file 
* View key and Download file 
* Logout

3.Test Authority

*Login the account
*Authorize Owner
*Authorize User
*Send key
*Logout

4.Cloud Server Provider

* Login the account 
* View all uploaded files
* Re-encryption request
* Graph
* Logout
